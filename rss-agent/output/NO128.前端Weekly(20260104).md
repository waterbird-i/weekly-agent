#### [【英文】Corundum – open-source FPGA-based NIC and platform for in-network compute](https://github.com/corundum/corundum)

Corundum 是一个开源的基于 FPGA 的网络接口卡（NIC）项目，同时也是一个网络内计算平台。它允许开发者在 FPGA 硬件上实现高性能网络功能和自定义数据包处理逻辑，支持网络内计算场景。该项目为研究人员和工程师提供了灵活的硬件网络开发平台，可用于网络加速、低延迟通信等前沿应用场景。

#### [【英文】Gershwin-desktop: OS X-like Desktop Environment based on GNUStep](https://github.com/gershwin-desktop/gershwin-desktop)

Gershwin-desktop 是一个基于 GNUStep 框架构建的桌面环境项目，旨在重现经典 Mac OS X 的视觉风格和用户体验。该项目为 Linux 用户提供了一种怀旧且优雅的桌面选择，融合了 NeXTSTEP/OpenStep 的设计哲学。对于喜爱苹果早期桌面美学或对 GNUStep 生态感兴趣的开发者来说，这是一个值得关注的开源项目。
